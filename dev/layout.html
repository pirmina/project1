1. Basic

  • html5로 작성된 ie7~최신 브라우저, chrome 크로스브라우징 반응형 사이트입니다.
  • lang = "en" 외국어사이트 입니다.(일부 컨텐츠에서 중국어, 일본어, 에스파뇰 포함합니다.)
  • DOCTYPE 아래 if조건문 주석으로 ie7, ie8일때 클래스가 추가됩니다.
    • <!--[if IE 7]><html lang="en" class="ie7"><![endif]-->
    • <!--[if IE 8]><html lang="en" class="ie8"><![endif]-->
  • 메타태그 : http-equiv="Expires" content="-1" 사용으로 익스플로러에서 더욱 빠르게 로딩되도록 합니다.
  • 메타태그 - viewport : content="width=device-width, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0, user-scalable=no" 반응형으로 로딩되도록 합니다.
  • 메타태그 - description : 자세한 description 작성으로 페이지에 대한 짧은 설명을 제공하여 검색엔진 최적화합니다.
  • 메타태그 - og : 페이스북 활용태그입니다. (개발미적용시 삭제할 항목)
  • apple-touch-icon, icon 파비콘을 추가합니다. (파비콘 링크는 http://jtbc.joins.com/ 로 부터 받아옵니다.)
  • Register 페이지는 html에 클래스 .bg_colored 추가해주세요.
  • ie7 크로스브라우징 소스는 images/ie7_**.png, _ie_browser.less 파일에서 .ie7{*} 입니다.
  • 웹접근성 일부 준수하여 시멘틱한 코드로 작업되었습니다.
  • Less파일 스타일 작성시, PC 기준으로(_common.less) 작성한 후 _media.less 또는 미디어쿼리로 모바일 스타일 작성해주세요.
  • PC : 1025px 이상의 화면, 미디어쿼리 없이 작성.
  • 모바일 : @media all and (max-width:1024px)
  • 태블릿 : @media (min-width:768px) and (max-width:1024px)

3. spritesmith를 이용한 Retina images 함수 만들기 ( _mixin.less )
.sprite-retina-width(@sprite) {
	width: (extract(@sprite, 5)/2); // width값/2
}
.sprite-retina-height(@sprite) {
	height: (extract(@sprite, 6)/2); // height값/2
}
.sprite-retina-position(@sprite) {
	  @sprite-retina-offset-x: (extract(@sprite, 3)/2);
	  @sprite-retina-offset-y: (extract(@sprite, 4)/2);
	  background-position: @sprite-retina-offset-x @sprite-retina-offset-y; // background-position x,y값/2
}
.sprite-retina-background-size(@sprite){
	@sprite-retina-size-width:(@spritesheet-width/2);
	@sprite-retina-size-height:(@spritesheet-height/2);
	background-size: @sprite-retina-size-width @sprite-retina-size-height; // background-size x,y값/2
}
.sprite-retina(@sprite) {
	.sprite-image(@sprite); // sprite.png  이미지 불러오기,
	.sprite-retina-position(@sprite); // background-position 값 불러오기
	.sprite-retina-width(@sprite); //  width값 불러오기
	.sprite-retina-height(@sprite); // height값 불러오기
	.sprite-retina-background-size(@sprite); // background-size값 불러오기
}
